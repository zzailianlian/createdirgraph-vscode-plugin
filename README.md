## 成品图样式

```
├── .DS_Store
├── images
│   ├── .DS_Store
│   ├── bg.png
│   ├── icon_arrow.png
│   ├── icon_cancel.png
│   ├── icon_close.png
│   ├── icon_safe.png
│   ├── icon_trans.png
│   ├── icon_vip_activited.png
│   └── main_bg_pink.webp
├── index.less
└── index.vue
```



## 安装

1. 将目录的`.vsix`文件下载到本地
2. vscode中打开插件侧边栏
3. 点击`...`
4. 点击`从VSIX安装...`
5. 选择下载好的`.vsix`文件即可
